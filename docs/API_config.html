配置 API

配置 API

配置 API 都需要重新插入设备或调用 RX78::Device::Restart RX78设备.重启 后生效。

RX78::Device::ConfigVendorID RX78设备.配置厂商ID

配置整个设备的厂商ID，2 头和 3 头设备将把把所有头都设置为相同的ID。

参数 注释
vendor_ID厂商ID 新的ID。

RX78::Device::ConfigMouseVendorID RX78设备.配置鼠标厂商ID

仅 3 头设备有效。

RX78::Device::ConfigName RX78设备.配置名称

配置整个设备的名称，2 头和 3 头设备将把把所有头都设置为相同的ID。

参数 注释
device_name名称 新的设备名。名称长度要求大于 0 小于等于 29，一个中文符号也算一个长度。
