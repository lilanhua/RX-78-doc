配置 API

配置 API

配置 API 都需要重新插入设备或调用 RX78::Device::Restart RX78设备.重启 后生效。

RX78::Device::ConfigVendorID RX78设备.配置厂商ID

参数 注释
vendor_ID厂商ID 新的ID。

RX78::Device::ConfigProductID RX78设备.配置产品ID

参数 注释
vendor_ID产品id 新的ID。

RX78::Device::ConfigDeviceName RX78设备.配置设备名

参数 注释
device_name设备名 新的设备名。名称长度要求大于 0 小于 29，一个中文符号也算一个长度。
